# Changelog

All notable changes to this project will be documented in this file.

The format is based on [Keep a Changelog](https://keepachangelog.com/en/1.1.0/).

## [Unreleased]

## [May 2024]

### Added
- Drawing inside headline and subtext areas.
- Video playback for recorded movies.
- Screen recording.

### Fixed
- Recorder now releases properly after a session finishes.

## [August 2023]

### Added
- Initial release with drawing capabilities.
- Multiple canvas support.
- Headline and subtext with customisable colours.
- Light and dark mode support.
- Optional grid background.
- Custom colour palette.
- Fat pen and marker tools.
- Crop area for 1920x1080 video output.
- App icon.

### Fixed
- Correct primary ink colour when launching in dark mode.
- Fixed border colour display.
- Disabled scribble on text fields.
