# Changelog

All notable changes to this project will be documented in this file.

The format is based on [Keep a Changelog](https://keepachangelog.com/en/1.1.0/).

## [Unreleased]

## [May 2024]

### Added
- Drawing inside headline and subtext areas
- Video playback support for recorded movies
- Screen recording feature

### Fixed
- Recorder now properly releases after recording session finishes

## [August 2023]

### Added
- Multiple canvas support
- Headline and subtext feature with customizable colors
- Dark and light mode support
- Grid background option
- Custom color palette including yellow
- Fat pen and marker tools
- Vertical space for cropping canvas to 1920x1080 video
- App icon
- Initial implementation with drawing capabilities

### Changed
- Updated layout and UI refinements
- Removed global accent color

### Fixed
- Primary ink color when app launches in dark mode
- Border color display
- Disabled scribble on text fields
- Xcode folder references
